Agent Skeleton — Competitor Overnight Brief

Day 3 starting point. Plain Python + Claude API. No frameworks.

This is the full working skeleton for the prototype agent. The pipeline is: parse task → fetch pages → compare with stored state → synthesize brief → flag uncertainty. See the Prototype Plan for architecture context.

Component Overview

1. Context Store

JSON files, one per user. Stores company description, competitor page states (content hashes + previews), and previous briefs. No database needed — the file growing over time makes “memory” tangible for the demo.

2. Web Fetching

Uses Jina Reader (r.jina.ai) as a proxy — handles JS rendering and returns clean markdown. Falls back gracefully on fetch errors. One-second delay between requests.

3. Diffing

SHA-256 hash comparison to detect changes. When content differs from stored state, generates a unified diff of the first 2,000 characters. New pages (no previous state) are flagged accordingly.

4. LLM Calls (Claude)

Two LLM calls in the pipeline:

  • Task parsing: Breaks the monitoring request into structured sub-tasks (competitor name, page types, URLs, key concerns). Returns JSON.
  • Brief synthesis: Takes all fetched data + diffs + user context and produces the morning brief. Items below 80% confidence go into uncertain_items instead of changes.

5. Execution Log

Timestamped step-by-step log. Designed to be shown in the UI as a “what the agent did overnight” replay. Prints to stdout during execution.

6. Main Pipeline

Orchestrates everything: load context → parse task → resolve URLs → fetch pages → diff against stored state → synthesize brief → update context store → write output JSON.

Full Source

Dependencies: pip install httpx anthropic. Requires ANTHROPIC_API_KEY env var.

Usage: python agent_skeleton.py "Monitor Linear -- track pricing, blog, changelog, careers"

#!/usr/bin/env python3
"""
Launch Agent: Competitor Overnight Brief
Sprint prototype — Day 3 starting point.

Architecture: Plain Python + Claude API. No frameworks.
Pipeline: Parse task → Fetch pages → Compare with stored state → Synthesize brief → Flag uncertainty

Usage:
    python agent_skeleton.py "Monitor Linear -- track pricing, blog, changelog, careers"
"""

import json
import sys
import time
import hashlib
import difflib
from datetime import datetime
from pathlib import Path

import httpx  # pip install httpx
import anthropic  # pip install anthropic

# --- Config ---
CLAUDE_MODEL = "claude-sonnet-4-20250514"
CONTEXT_DIR = Path("context_store")
JINA_PREFIX = "https://r.jina.ai/"  # free, handles JS rendering, returns markdown

# --- Context Store (JSON files, one per user) ---

def load_context(user_id: str = "demo_user") -> dict:
    path = CONTEXT_DIR / f"{user_id}.json"
    if path.exists():
        return json.loads(path.read_text())
    return {
        "user_id": user_id,
        "company_description": "",
        "competitors": {},  # competitor_name -> {pages: {url: last_content_hash}, last_brief: str}
        "briefs": [],
    }

def save_context(ctx: dict):
    CONTEXT_DIR.mkdir(exist_ok=True)
    path = CONTEXT_DIR / f"{ctx['user_id']}.json"
    path.write_text(json.dumps(ctx, indent=2, default=str))


# --- Web Fetching ---

def fetch_page(url: str) -> str:
    """Fetch a page via Jina Reader (handles JS, returns markdown)."""
    try:
        resp = httpx.get(f"{JINA_PREFIX}{url}", timeout=30, follow_redirects=True)
        resp.raise_for_status()
        return resp.text
    except Exception as e:
        return f"[FETCH_ERROR: {e}]"

def fetch_competitor_pages(competitor: str, urls: list[str]) -> dict[str, str]:
    """Fetch all pages for a competitor. Returns {url: content}."""
    results = {}
    for url in urls:
        print(f"  Fetching: {url}")
        results[url] = fetch_page(url)
        time.sleep(1)  # be polite
    return results


# --- Diffing ---

def compute_hash(content: str) -> str:
    return hashlib.sha256(content.encode()).hexdigest()[:16]

def compute_diffs(old_pages: dict, new_pages: dict[str, str]) -> dict[str, str]:
    """Compare new page contents against stored state. Returns {url: diff_summary}."""
    diffs = {}
    for url, new_content in new_pages.items():
        old_hash = old_pages.get(url, {}).get("hash", "")
        new_hash = compute_hash(new_content)
        if old_hash != new_hash:
            old_content = old_pages.get(url, {}).get("content_preview", "")
            # Compute a readable diff (first 2000 chars of each)
            diff = difflib.unified_diff(
                old_content[:2000].splitlines(),
                new_content[:2000].splitlines(),
                lineterm="",
                fromfile="previous",
                tofile="current",
            )
            diffs[url] = "\n".join(diff) if old_hash else "[NEW PAGE - no previous state]"
    return diffs


# --- LLM Calls ---

def call_claude(system: str, user: str) -> str:
    client = anthropic.Anthropic()  # uses ANTHROPIC_API_KEY env var
    resp = client.messages.create(
        model=CLAUDE_MODEL,
        max_tokens=4096,
        system=system,
        messages=[{"role": "user", "content": user}],
    )
    return resp.content[0].text


def parse_task(task_description: str) -> dict:
    """Break a monitoring request into structured sub-tasks."""
    result = call_claude(
        system="You break monitoring requests into structured plans. Return JSON only.",
        user=f"""Given this monitoring request, identify:
1. The competitor name
2. Which pages to check (pricing, blog, changelog, careers, etc.)
3. Specific URLs if inferable, otherwise leave blank
4. What aspects the user cares about most

Request: {task_description}

Return JSON: {{"competitor": str, "pages": [{{"type": str, "url": str, "focus": str}}], "key_concerns": [str]}}"""
    )
    # Parse JSON from response (handle markdown code blocks)
    result = result.strip()
    if result.startswith("```"):
        result = result.split("\n", 1)[1].rsplit("```", 1)[0]
    return json.loads(result)


def synthesize_brief(
    competitor: str,
    diffs: dict[str, str],
    page_contents: dict[str, str],
    user_context: dict,
    key_concerns: list[str],
) -> dict:
    """Generate the morning brief from all collected data."""

    diff_summary = "\n\n".join(
        f"### {url}\n{diff}" for url, diff in diffs.items()
    ) if diffs else "No changes detected from previous state."

    content_summary = "\n\n".join(
        f"### {url}\n{content[:1500]}..." for url, content in page_contents.items()
    )

    result = call_claude(
        system="""You are an analyst producing overnight competitor briefs.
Your output is a JSON object with this structure:
{
  "summary": "1-2 sentence executive summary",
  "changes": [{"title": str, "description": str, "significance": "high|medium|low", "recommended_action": str}],
  "uncertain_items": [{"title": str, "description": str, "why_uncertain": str, "question_for_user": str}],
  "no_change_areas": [str]
}

Be specific and actionable. If you're less than 80% confident in a conclusion, put it in uncertain_items instead of changes. It's better to flag uncertainty than to be wrong.""",
        user=f"""Competitor: {competitor}

User's company context:
{json.dumps(user_context, indent=2)}

Key concerns: {', '.join(key_concerns)}

Changes detected (diffs from last check):
{diff_summary}

Current page contents:
{content_summary}

Produce the overnight brief."""
    )

    result = result.strip()
    if result.startswith("```"):
        result = result.split("\n", 1)[1].rsplit("```", 1)[0]
    return json.loads(result)


# --- Execution Log ---

class ExecutionLog:
    def __init__(self):
        self.entries = []

    def log(self, step: str, detail: str = ""):
        ts = datetime.now().strftime("%H:%M:%S")
        entry = {"time": ts, "step": step, "detail": detail}
        self.entries.append(entry)
        print(f"[{ts}] {step}" + (f" — {detail}" if detail else ""))

    def to_dict(self):
        return self.entries


# --- Main Pipeline ---

def run_agent(task_description: str, user_context: dict = None):
    log = ExecutionLog()
    ctx = load_context()

    if user_context:
        ctx["company_description"] = user_context.get("company_description", "")

    # Step 1: Parse task
    log.log("Parsing task", task_description)
    plan = parse_task(task_description)
    competitor = plan["competitor"]
    log.log("Identified competitor", competitor)
    log.log("Pages to check", ", ".join(p["type"] for p in plan["pages"]))

    # Step 2: Resolve URLs (fill in any blanks)
    urls = []
    for page in plan["pages"]:
        if page["url"]:
            urls.append(page["url"])
        else:
            # Could do a search here, or use known URL patterns
            log.log("Skipping page (no URL)", page["type"])

    if not urls:
        log.log("ERROR", "No URLs to fetch. Provide explicit URLs or improve URL resolution.")
        return None

    # Step 3: Fetch pages
    log.log("Fetching pages", f"{len(urls)} pages")
    page_contents = fetch_competitor_pages(competitor, urls)
    fetched = sum(1 for v in page_contents.values() if not v.startswith("[FETCH_ERROR"))
    log.log("Fetched successfully", f"{fetched}/{len(urls)}")

    # Step 4: Compare with stored state
    log.log("Comparing with previous state")
    old_pages = ctx.get("competitors", {}).get(competitor, {}).get("pages", {})
    diffs = compute_diffs(old_pages, page_contents)
    log.log("Changes detected", f"{len(diffs)} pages changed")

    # Step 5: Synthesize brief
    log.log("Synthesizing brief")
    brief = synthesize_brief(
        competitor=competitor,
        diffs=diffs,
        page_contents={k: v for k, v in page_contents.items() if not v.startswith("[FETCH_ERROR")},
        user_context={"company": ctx.get("company_description", ""), "previous_briefs": ctx.get("briefs", [])[-3:]},
        key_concerns=plan.get("key_concerns", []),
    )
    log.log("Brief generated", f"{len(brief.get('changes', []))} changes, {len(brief.get('uncertain_items', []))} uncertain")

    # Step 6: Update context store
    if competitor not in ctx.get("competitors", {}):
        ctx.setdefault("competitors", {})[competitor] = {"pages": {}}
    for url, content in page_contents.items():
        if not content.startswith("[FETCH_ERROR"):
            ctx["competitors"][competitor]["pages"][url] = {
                "hash": compute_hash(content),
                "content_preview": content[:2000],
                "fetched_at": datetime.now().isoformat(),
            }

    result = {
        "competitor": competitor,
        "brief": brief,
        "execution_log": log.to_dict(),
        "generated_at": datetime.now().isoformat(),
    }
    ctx.setdefault("briefs", []).append(result)
    save_context(ctx)

    # Step 7: Write output
    output_path = Path(f"briefs/{competitor}_{datetime.now().strftime('%Y%m%d_%H%M%S')}.json")
    output_path.parent.mkdir(exist_ok=True)
    output_path.write_text(json.dumps(result, indent=2, default=str))
    log.log("Brief saved", str(output_path))

    return result


if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: python agent_skeleton.py 'Monitor [competitor] -- track ...'")
        sys.exit(1)

    task = " ".join(sys.argv[1:])

    # Demo user context — customize for the demo
    demo_context = {
        "company_description": "Felix is a proactive AI personal assistant. iOS app with swipeable action cards. Key differentiator: agents that work while you sleep.",
    }

    result = run_agent(task, demo_context)

    if result:
        print("\n" + "=" * 60)
        print("OVERNIGHT BRIEF")
        print("=" * 60)
        brief = result["brief"]
        print(f"\n{brief.get('summary', 'No summary')}\n")

        for change in brief.get("changes", []):
            sig = change.get("significance", "?").upper()
            print(f"[{sig}] {change['title']}")
            print(f"  {change['description']}")
            print(f"  → {change['recommended_action']}\n")

        if brief.get("uncertain_items"):
            print("--- NEEDS YOUR INPUT ---")
            for item in brief["uncertain_items"]:
                print(f"? {item['title']}")
                print(f"  {item['description']}")
                print(f"  {item['question_for_user']}\n")

Key Design Decisions

  • No agent frameworks. Plain Python + Claude API. LangChain and CrewAI add abstractions you’ll fight during debugging. Refactor later.
  • JSON file context store. One file per user. Readable, inspectable, trivially debuggable. You can literally show the file growing over time in the demo.
  • Jina Reader for web fetching. Handles JS rendering without running a headless browser. Free tier is enough for the demo.
  • Two LLM calls, not ten. One to parse the task into structure, one to synthesize the brief. Minimizes latency and cost.
  • 80% confidence threshold. The synthesis prompt explicitly tells Claude to route low-confidence conclusions into uncertain_items. This is how the “trust moment” in the demo gets generated.
  • Hardcoded pipeline. This agent does ONE thing: competitor monitoring. General-purpose routing is a post-demo concern.

Running It

  1. Install deps: pip install httpx anthropic
  2. Set your API key: export ANTHROPIC_API_KEY="sk-..."
  3. Run: python agent_skeleton.py "Monitor Linear -- track pricing, blog, changelog, careers"
  4. Output goes to briefs/ directory as timestamped JSON.
  5. Run it again to see diffs — the context store remembers the previous state.
