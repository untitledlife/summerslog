Track 08: Launch Agents

No solutions. No tech. No features. Human pain only.

Gate 1 Checkpoint · Tuesday 4 PM
1.1 — Problem Statement

The glue is a person

Founders and operators struggle to find time for high-value work because they're stuck doing low-value tasks that still need to get done.
  • AI tools exist but the workflow is fragmented across 5-8 apps
  • Tasks keep piling up because nothing holds enough context to take them off the plate
  • The founder becomes the system that never turns off
1.2 — Current Workaround

How they cope today

  • Do things manually — the default. 73% of founders report persistent burnout from doing everything themselves
  • Use AI tools, but fragmented — Claude for thinking, ChatGPT for voice, Notion for storage, Chronicle for decks. Average worker switches apps 1,200x/day
  • Delegate to humans — VAs ($1-4K/month), freelancers, interns. 38% VA turnover in year one. 4-6 weeks to onboard each time
  • Try agent platforms — Lindy, Manus, Devin. Demo-to-production gap is massive. "Like having a toddler that needs to be overseen"

All fail at the same boundary: where execution requires judgment + context.

1.3 — Cost of the Problem

What it actually costs

Time
10-30 hrs/week
Spent on tasks that should be delegable. 1+ hr/day just on context switching between tools.
Emotional
Daily stress
73% burnout rate. 68% chronic sleep disruption. 49% considered quitting in the past year.
Opportunity
A lot
Effective delegators see 100+ percentage points more growth. But 75% of founders struggle to delegate.
1.4 — Riskiest Assumption

What must be true

Assumption #1
Users will trust the platform enough to let it read their context and write actions on their behalf.
Today: 83% want human-in-the-loop. Only 17% trust AI to act autonomously. But trust doubles over ~750 sessions (Anthropic data).
Assumption #2
The agent delivers enough value that it becomes sticky, not just a novelty.
If tasks aren't done correctly (the way the user would've done them), they'll stop delegating and go back to doing it themselves. The 80% bar, not perfection.
